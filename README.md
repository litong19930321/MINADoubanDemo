# MINADoubanDemo
一个简单的豆瓣电影的demo
过完十一回来，发现微信开放了小程序开发文档，已经不用破解即可使用的开发工具，准备继续研究小程序，下面就是一个简单的豆瓣电影小程序的demo

程序分为四个模块
1. 电影推荐
2. 北美票房
3. 搜索模块
4. 我的简介
